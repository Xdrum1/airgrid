AirIndex API

Programmatic access to UAM market readiness ratings across 20+ U.S. cities. Seven-factor methodology, primary government data sources, real-time updates.

AIRINDEX
airindex.io/api
hello@airindex.io

Endpoints

GET /api/v1/markets

Current readiness ratings for all scored U.S. markets

GET /api/v1/markets/{city_id}

Full detail — operators, corridors, tier history, source citations

GET /api/v1/markets/{city_id}/history

Timestamped score history with triggering event attribution

GET /api/v1/markets/export

Bulk export — all markets at current ratings, pipeline-ready

Example

# Get all ADVANCED-tier markets
curl https://airindex.io/api/v1/markets \
-H "Authorization: Bearer aix_your_key"
// Response envelope
{
"meta": { "total": 21, "methodology": "v1.0" },
"data": [
{ "city_id": "dallas", "score": 100, "tier": "ADVANCED", ... }
]
}

Authentication

All requests require a Bearer token in the Authorization header. Keys are issued from the AirIndex dashboard.

Authorization: Bearer aix_xxxxxxxxxxxx

Keys are prefixed with aix_. SHA-256 hashed at rest. Revocable from dashboard.

Response Format

Format JSON — all responses
Keys snake_case throughout
Timestamps ISO-8601 (UTC)
Envelope { meta, data }
Market IDs snake_case slugs
Headers X-RateLimit-* on every response

Rate Limits

Pro 100 requests / hour
Institutional 1,000 requests / hour
Enterprise 10,000 requests / hour

429 responses include Retry-After header. Rate limits enforced via Upstash Redis.
