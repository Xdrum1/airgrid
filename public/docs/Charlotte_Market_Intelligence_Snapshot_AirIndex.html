WHAT IS HOLDING CHARLOTTE'S SCORE DOWN

AirIndex runs a five-question ordinance audit for each market to assess whether the city's legal and regulatory framework supports vertical flight infrastructure. Charlotte has four gaps — all of which are within the city's direct control to fix. None require state legislation or private capital investment.

Question Status What this means
1. Correct FAA terminology in city code (heliport, helistop, vertiport per 14 CFR Part 157) NO Charlotte's UDO (updated Dec 2025) contains no vertiport terminology. NC is developing an AAM strategy but the city code has not been updated. Using undefined terms like "helipad" creates legal ambiguity in any permit application.
2. Heliport/vertiport addressed as permitted or conditional use in zoning ordinance NO Charlotte UDO (eff. June 2023, updated Dec 2025) does not address vertiports. No clear zoning pathway for vertiport development. Developers cannot determine where a vertiport can be built without a special exception process.
3. FAA airspace determination required in permit process NO No evidence of FAA airspace determination in Charlotte's permit process. Without this requirement, heliports and vertiports can be approved without confirming FAA has reviewed the airspace impact — creating liability exposure for the city.
4. NFPA 418 referenced in fire or building code PARTIAL NC adopted IBC, which references NFPA 418 for rooftop heliports. However, Charlotte's standalone fire code does not explicitly reference NFPA 418 for all vertical flight infrastructure.
5. State (NC) has adopted FAA AC 150/5390-2D as enforceable standard NO North Carolina has not yet adopted the FAA heliport design standard as enforceable at the state level. The Charlotte-Atlanta corridor announcement signals this is in motion but not yet law.

■ 90-Day Window: The FAA is expected to publish a unified vertical flight infrastructure Advisory Circular for public comment around June 2026 — the merger of AC 150/5390-2D (Heliport Design) and Engineering Brief 105A (Vertiport Design). Cities that update their ordinance language before this publication will be positioned to align with the new federal standard from day one. Cities that wait will face a more complex update process after the fact.
